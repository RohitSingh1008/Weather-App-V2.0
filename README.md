# 🌦️ Weather App V2.0

## 📖 Description
A real-time weather web application developed using HTML, CSS and JavaScript with auto-location detection, API integration, weather-based sound effects and voice feedback support. The app automatically detects the user's current location and displays live weather updates with dynamic weather sounds and manual city search functionality.

---

## ✨ Features
- 📍 Auto-location weather detection
- 🔍 Manual city weather search
- 🌦️ Real-time weather updates
- 🔗 Weather API integration
- 🎤 Voice feedback support
- 🔊 Weather-based sound effects
- 🌩️ Thunder, rain and wind sounds
- 📱 Responsive design
- ⚡ Interactive UI

---

## 🛠 Technologies Used
- HTML
- CSS
- JavaScript
- Weather API
- Geolocation API
